About This App

Welcome to our Calendar App, designed to help you manage your time effectively. This application allows you to effortlessly add, delete, and move appointments, providing a seamless scheduling experience.

Features

  • Add Appointments: Click on any date to add a new appointment with a title and description.
  • Delete Appointments: Remove unwanted appointments by clicking the delete button on the appointment.
  • Move Appointments: Drag and drop appointments to reschedule them to a different date.
  • Monthly Navigation: Easily navigate between months using the previous and next buttons.
  • Yearly Navigation: Move between years by navigating through months.
  • Customizable Start Day: The calendar can start from any day of the week you prefer.
  • Persistent Data: All appointments are saved in your local storage, ensuring your data is retained even after refreshing the page.

How to Use

  1. Select a date to view or add appointments.
  2. Use the input fields to enter your appointment details.
  3. Click the save button to store your appointment.
  4. Drag and drop appointments to reschedule them as needed.
  5. Navigate between months and years using the navigation buttons.

Technology Stack

This app is built using the following technologies:

  • Angular: A robust framework for building dynamic web applications.
  • Angular Material: A UI component library for Angular that implements Google's Material Design.
  • Angular CDK: A set of behavior primitives for building complex UI components.
  • TypeScript: A typed superset of JavaScript that compiles to plain JavaScript.

Achievements

  • Dependency Injection: Utilized Angular's Dependency Injection to manage service dependencies, such as in the AppointmentService and AppointmentDialogComponent.
  • Lazy Loading: Implemented lazy loading for modules and routes to optimize application performance.
  • Routing: Incorporated Angular's RouterModule and RouterOutlet for navigating between different components.
  • Angular Forms: Employed reactive forms with validators and value changes in the AppointmentDialogComponent for managing appointment data.
  • RXJS: Used RXJS for managing state and asynchronous operations, such as in the AppointmentService with BehaviorSubject.
  • Standalone and Shared Components: Organized components into standalone and shared modules for better modularity and reusability.
  • Angular Material and CDK: Leveraged Angular Material for consistent styling and Angular CDK for drag-and-drop functionality.
